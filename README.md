# Medical Insurance Data Analysis

## Project Overview

This project focuses on analyzing a medical insurance dataset using SQL to extract meaningful insights such as cost patterns, demographics impact, and risk factors.

The goal is to demonstrate data analysis skills including querying, data cleaning, and generating business insights.

---

## Dataset Information

Dataset Name: Medical Insurance Dataset
Features:
Age
Gender
BMI
Number of Children
Smoker Status
Region
Charges (Medical Cost)

---

## Tools & Technologies

SQL (MySQL / PostgreSQL / SQLite)
Database Management Systems
Data Analysis Techniques

---

## Key Analysis Performed

🔹 Average insurance charges by age group
🔹 Impact of smoking on medical costs
🔹 BMI vs insurance charges analysis
🔹 Regional cost comparison
🔹 Gender-based cost insights

---

## Insights

Smokers have significantly higher insurance charges compared to non-smokers
Higher BMI tends to increase medical expenses
Certain regions show higher average costs
